This project is a space to learn Python for IAM tasks. Each day interates introduces a new concept to practice with a series of tasks. Days interate up to a weekly milestone project.

Tools:
Day_06.py is a Stale Account Detector that takes in a .csv of user data and outputs a list of stale users and saves it to csv. To run the tool, run the script name with an optional arguemnt for threshold in days for an account to be considered stale.

For example. "day_06.py 30" detects accounts that have not been used in 30 days.
